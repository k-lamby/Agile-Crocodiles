This web application was created by Katherine Lambert for the UOL course databases, networks and the web.

I wanted to produce it with a bold simple style that aligns with many existing health apps out there. I used a data visualisation colour palette picker to set a colour scheme with high contrast for accessibility.

When designing this app it was important to make navigation intuitive and logical for the end user.
It has been built using ejs, express and mysql.

The mysql database contains a single table which contains a basic dataset from myfooddata.com of around 1000 items.
